Weather Dashboard
repository

This application enables a user to search for any city and get its main weather components for the current date and time, which is shown at the top of the page. The app also shows the city's five-day forecast. Searches are saved to local storage so they can be called up again easily.


Created using HTML5/CSS and JavaScript.

Day Planner
repository

This app helps organize a day's events. A user can save daily plans based on the hours of the day. The colors of the boxes change based on if the event is present, past, or upcoming. Events are saved to local storage so they can be called up again easily. There is a "clear" button at the bottom of the page that erases all entries. The header area features a current clock.

Created using HTML5/CSS and JavaScript.
